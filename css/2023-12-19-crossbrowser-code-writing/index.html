Tarayıcılar arası tutarlı kod yazmak için:

  1. Tarayıcı varsayılı ( default ) stillerinin farkında olmak. Gerektiğinde bu stilleri tüm tarayıcılar için aynı zemine taşımak. Reset kullanmak.
  2. Kullandığınız css özelliklerinin tarayıcı uygunluğunu kontrol edin. Cross browser compatibility Can I use? sitesinden bakabilirsiniz...
  3. Gerekli olduğunda vendor prefix ( tarayıcı öneki kullanın )
    • chrome, safari prefix: -webkit-
    • firefox: -moz-
    • opera: -o-
    • edge: -ms-
  4. css @support media kuralı ile de uygunluk kontrolü ( compatibility check ) yapabilrsin
  5. firefox, opera, edge, safari gibi temel tarayıcılar ile testler yapabilirsiniz...
